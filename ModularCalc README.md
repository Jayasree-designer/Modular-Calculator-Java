# Modular Calculator

## Description
This is a simple Java console application.
The program performs basic arithmetic operations
using separate methods in a modular way.
It also demonstrates method overloading,
exception handling, and pass-by-value concept.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Method overloading
- Divide-by-zero handling
- Pass-by-value demonstration

## Concepts Used
- Methods (functions)
- Method overloading
- if condition
- try-catch (exception handling)
- Scanner class
- Pass-by-value
- Console input and output

## Files Included
- ModularCalculator.java
- output.png (program output screenshot)

## How to Run
1. Compile the program:
   javac ModularCalculator.java

2. Run the program:
   java ModularCalculator

## Output
The program displays:
- Results of arithmetic operations
- Exception message for divide by zero
- Pass-by-value behavior  
Output screenshot is included in the repository.
